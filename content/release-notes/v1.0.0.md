## Modes

- 14 analysis modes: Write, Rewrite, Review, Style Check, Simplify, Keywords, Glossary, Code to Docs, User Guide, Explain, Summarize, Expand, Translate, Outline
- Each mode uses a purpose-built system prompt tailored to its task

## Style enforcement

- 25 default style rules covering voice, tense, terminology, conciseness, structure, and formatting
- Custom style rules (Pro tier)
- 3 default terminology substitutions (select/click, enter/type in, repository/repo)
- Product glossary with definitions and synonyms — the AI uses your definitions when writing

## Readability

- Live readability scores: Flesch Reading Ease, Flesch-Kincaid Grade Level, Gunning Fog Index
- Before/after comparison in Rewrite and Simplify modes
- Word-level diffs showing exactly what changed

## Workspace

- Context bar with Audience, Document Type, and Mode selectors
- File attachments for Code to Docs mode
- Session history with mode, audience, and timestamp
- Copy output to clipboard

## Configuration

- Import/export settings as JSON
- Custom AI instructions
- Toggle individual style rules on/off

## Auth and billing

- Clerk authentication (sign up, sign in)
- Free tier: 10 requests/month, 3 glossary terms, 3 terminology substitutions
- Pro tier: unlimited requests, custom rules, unlimited glossary and terminology
- Stripe checkout and billing portal

## Documentation

- In-app docs at /docs with Getting Started, Modes, Settings, Readability, and Pricing pages
