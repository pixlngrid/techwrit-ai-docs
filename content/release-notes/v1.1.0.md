## Prompt Library

- 4 built-in prompts: Feature Requirements Doc, Release Notes, API Endpoint Doc, Troubleshooting Guide
- Prompt Library dropdown in the context bar pre-fills the input area with structured prompts containing placeholders
- Prompts auto-set mode and doc type when applicable
- Create, edit, and delete custom prompts in Settings > Prompt Library
- Custom prompts persist locally and sync to the server
- Prompts included in configuration export/import

## Workspace and settings

- Download output as Markdown from the output panel
- New Prompt Library tab in Settings for managing custom prompts
- Settings dialog updated to 5-tab layout (Style Rules, Terminology, Glossary, Prompt Library, General)
