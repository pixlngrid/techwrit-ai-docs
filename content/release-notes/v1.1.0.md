## Prompt templates

- 4 built-in templates: Feature Requirements Doc, Release Notes, API Endpoint Doc, Troubleshooting Guide
- Templates dropdown in the context bar pre-fills the input area with structured prompts containing placeholders
- Templates auto-set mode and doc type when applicable
- Create, edit, and delete custom templates in Settings > Templates
- Custom templates persist locally and sync to the server
- Templates included in configuration export/import

## Workspace and settings

- Download output as Markdown from the output panel
- New Templates tab in Settings for managing custom templates
- Settings dialog updated to 5-tab layout (Style Rules, Terminology, Glossary, Templates, General)
