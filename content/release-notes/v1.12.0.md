## Word document upload

- **Upload `.docx` files** directly using the paperclip icon — Word documents are converted to Markdown on upload, preserving headings, lists, tables, and formatting
- Import content from Confluence (export to Word), Google Docs, or Microsoft Word for review, rewrite, or any other mode
- Works with all modes: upload a Word doc in Review mode, then switch to Rewrite without re-uploading
- Mixed uploads supported: attach `.docx` files alongside code, Markdown, and other text files in the same session
- Batch processing works with `.docx` files — upload multiple Word documents and process each separately or analyze them together

## Persistent file attachments

- Attached files now persist after submission — switch modes (e.g., Review to Rewrite) without losing your uploaded files
- Files are cleared on Reset or New Chat, or removed individually with the X button

## Prompt library rename

- **Templates** renamed to **Prompt Library** across the app — the Prompt Library dropdown, Settings tab, and all related UI
- Individual items are now called "prompts" instead of "templates"
- Existing configurations are preserved — no action needed

## Docs site

- Documentation moved to [docs.techwrit.ai](https://docs.techwrit.ai)
- Sidebar Docs link now opens the external docs site
