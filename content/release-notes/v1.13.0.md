## Diagram export — PNG and Excalidraw

- **PNG export** — download any generated Mermaid diagram as a high-resolution PNG image, ready to embed in docs, Confluence, Slack, or anywhere images are accepted
- **Excalidraw export** — download diagrams as editable `.excalidraw` files to customize shapes, text, arrows, and layout in Excalidraw (web, desktop, or VS Code extension)
- Both export buttons appear in the diagram header bar alongside the existing Hide/Show toggle
- Exports are lazy-loaded — the conversion libraries only download when you click an export button, keeping the app fast
