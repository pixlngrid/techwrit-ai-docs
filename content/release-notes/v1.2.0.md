## Custom AI instructions per mode

- Set different custom instructions for each of the 14 modes instead of a single global instruction
- Mode-specific instructions replace the global default for that mode; modes without custom instructions fall back to the global instruction
- New mode selector dropdown in Settings > General lets you pick "Global (default)" or any specific mode to edit its instruction
- Modes with custom instructions are marked with a dot indicator in the dropdown
- Per-mode instructions are included in configuration export/import and sync to the server

## Improved review accuracy

- Review mode no longer flags text that already complies with the rules
- The AI must now show a concrete before/after difference for every issue it reports, reducing false positives
