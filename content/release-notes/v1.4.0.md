## Team plans

- Team tier with shared style rules, terminology, and glossary enforced across all members
- Invite-code system for adding team members (up to seat limit)
- Team management dialog with member list, role management, and shared config editor
- Team config merges additively — team items are locked, members add personal items on top
- Per-seat billing via Stripe (monthly or annual)

## History search and filtering (Pro/Team)

- Search across input and output previews to find past analyses
- Filter by mode (Write, Rewrite, Review, Simplify, Code to Docs)
- Filter by date range (last 7, 30, or 90 days)
- Result count shown when filters are active
- Output preview now stored for better search coverage
