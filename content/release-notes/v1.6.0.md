## VS Code extension (Pro/Team)

- Review, rewrite, and simplify Markdown documents directly in VS Code
- Review and Style Check results appear as native VS Code diagnostics (squiggly underlines with quick-fix suggestions)
- Rewrite and Simplify update the document in place (Ctrl+Z to undo)
- Selection support: review or rewrite just the highlighted text
- Keyboard shortcuts: Ctrl+Shift+F6 (Review), F7 (Style Check), F8 (Rewrite), F9 (Simplify)
- Optional auto-review on save for Markdown files
- Uses your saved style rules, terminology, and glossary from your TechWrit AI account
- Shared quota with the web app — no separate billing
- Install from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=techwrit.techwrit-ai)
